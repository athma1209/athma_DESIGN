Cherry Quant의 기본 가이드 전략은 Docker에 등록되어 있으며, 제공되는 전략을 바탕으로 나만의 전략을 만들 수 있습니다

Docker + Vs Studio Code

전략을 작성하기 위해 ‘docker’와 ‘visual studio code’ 프로그램을 설치해주세요

Cherry Quant의 전략 개발 언어는 Python으로 진행됩니다.

보편적으로 Quant Algorithmic Trading의 전략 개발시 Python이 사용되고 있습니다. 앞으로 저희 Cherry Quant는 전략 고도화 및 업데이트를 통해 Python 외 다양한 환경에서의 전략 개발을 제공을 하기위해 준비하고 있습니다

Simulation

개발된 전략은 ‘Backtesting’, ‘Forward Testing’을 통해 수익률을 검증할 수 있으며, Cherry Quant에서 기본으로 제공하는 Simulaton View Page로 전략 결과쉽게 확인 할 수 있습니다.

추후 업데이트 후 원하시는 형태로 커스텀 가능한 소스를 제공할 예정입니다
